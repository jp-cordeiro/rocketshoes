- Tecnologias utilizadas:
* React
* Redux
* Redux-sagas

- Para executar o projeto:
```bash

# Instalar dependências
$ yarn install

# Iniciar o servidor em outra janela de terminal
$ yarn api:server

# Execute a aplicação
$ yarn start
```
